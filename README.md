<h3 align="center">Stolen Cars - API</h3>

<h4>Objectives</h4>

Facilitate the search of vehicles that have been stolen, could be SUV, Sports, Truck, Motorcycle. To insert a vehicle you must have the date of the theft, the color of the vehicle and the model

<h4>Features</h4>
- [x] API
    - [x] Add stoled vehicle
    - [x] Delete stoled vehicle
    - [x] List all stoled vehicles
    - [x] Search vehicle (id based)
    - [ ] Search vehicle (model based)
    - [ ] Search vehicle (color based)
    - [ ] Search vehicle (chassis based)
