<h3 align="center">Stolen Cars - API</h3>

<h4>Objectives</h4>

Facilitate the search of vehicles that have been stolen, could be SUV, Sports, Truck, Motorcycle. To insert a vehicle you must have the date of the theft, the color of the vehicle and the model

<h4>Features</h4>
<ul>
    <li>API</li>
    <ul>
        <li><input type="checkbox" checked> I have a bike<br></li>
        <li><input type="checkbox" checked>Add stoled vehicle<br></li>
        <li><input type="checkbox" checked>Delete stoled vehicle<br></li>
        <li><input type="checkbox" checked>List all stoled vehicles<br></li>
        <li><input type="checkbox" checked>Search vehicle (id based)<br></li>
        <li><input type="checkbox">Search vehicle (model based)<br></li>
        <li><input type="checkbox">Search vehicle (color based)<br></li>
        <li><input type="checkbox">Search vehicle (chassis based)<br></li>
    </ul>
    <li>Interface</li>
</ul>
